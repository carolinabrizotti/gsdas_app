## Welcome!

This tutorial is designed to guide you through the features and information layers of this application.

This guide aims to clearly present the available resources, the paths to access the data, and the procedures for interacting with the 2D and 3D visualizations. 

The application was developed with the intention of providing the user with spatio-temporal information on the availability of groundwater in the Paranapanema River Basin. It provides information on total groundwater storage values ‚Äã‚Äãin millimeters (mm) throughout the period from 2003 to 2025, and statistics associated with the analysis of emerging hot/cold spots and trends in the basin.

Explore the following sections to understand the content structure and unlock the full potential of this tool.

#### 1. Interface Overview

Upon entering the application, use the expandable panels in the left sidebar to access comprehensive project information. This section provides details information regarding data methodology, map layers, and associated works (Figure 1)

#### 2. Navigation and Control Tools

The maps feature navigation tools located within the interface:

Zoom (In/Out): Controls the magnification level, also accessible via the mouse scroll wheel.

Default Extent (Home): Returns to the map‚Äôs initial view.

3D Navigation (Pan/Rotate): Toggles between panning and rotating the view. This feature is exclusive to the 3D map and can be controlled using the left and right mouse buttons, respectively.

Compass: Resets the map orientation to North.

Regarding data visualization, you can use the default interface with minimized windows or expand either map to full-screen mode. To do so, click the Maximize button (‚õ∂), located in the bottom-right corner of each map frame.
